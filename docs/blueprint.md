# **App Name**: Aura Learning: Gamified Studygram

## Core Features:

- Authentication: Simple email-based login to identify users locally; does not require password.  Data stored locally and keyed by email.
- Dashboard: Display of subject categories: Physics, Chemistry, Math, Biology, Coding, that a user can select to drill down and learn.
- Header Stats: Display User Name, Life Energy, Coin Balance, Streak Count, and available Streak Freezes to allow them to play and learn in an engaging way.
- Game/Quiz Level: Generates multiple-choice quizzes to support dynamic question generation for a topic.
- Life System: Deduct a 'heart' when a user gets a question wrong. Allow a heart to regenerate after a fixed period of time. Track hearts in local storage.
- Topic Selection: Provide a nested structure for a user to select, for example: Physics -> 'Basic Electrostatics'.  Show completion status, star rating and high score for each topic.
- Aura Store: Let a user spend earned coins. Includes life refill and streak freeze options. The user is shown the current coin count, which is retrieved from local storage.

## Style Guidelines:

- Primary color: Deep, rich dark blue (#00297A) to evoke a sense of depth and knowledge.
- Background color: Slightly darker shade of dark blue (#002166) to create a layered effect without harsh contrast.
- Accent color: Vibrant gold (#D1A226) for interactive elements and borders to signify achievement and prestige.
- Headline font: 'Cinzel', a serif font, for elegant, classic headlines.
- Body font: 'Lato', a sans-serif font, for clean, readable body text.
- Gold graduation cap (mortarboard) on a dark blue square background as the favicon, symbolizing academic achievement.
- A central, card-like container with large rounded corners and a glowing gold border, centered on the screen to draw focus.
